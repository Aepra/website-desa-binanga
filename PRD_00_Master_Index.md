# 📚 MASTER DOCUMENT: Smart Village Platform PRD Series

Dokumentasi ini adalah pusat dari perancangan arsitektur perangkat lunak untuk **Smart Village Platform**, didesain untuk menjadi portofolio berskala Enterprise (Startup-grade) bagi Full Stack/Data/Frontend Engineer.
